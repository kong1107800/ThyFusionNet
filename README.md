# ThyFusionNet
A CNN–Transformer Framework for Thyroid Disease Diagnosis
This project contains the model code, component code, training code, and other files related to the paper “ThyFusionNet: A CNN–Transformer Framework with Spatial Aware Sparse Attention for Multi-Modal Thyroid Disease Diagnosis.” It will be continuously updated.
